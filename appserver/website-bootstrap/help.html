About Twitter Trends

Twitter Trends is a web application that allows users to visualize and explore pertinent topics on the social media platform Twitter. The user can access an interface that offers options of fun and interactive visualizations and can switch between these views, including Locations, Time, and Network. When chosen, each view will present real time Twitter data with the focus on different aspects. The user does not need their own Twitter account to use this product; however, they have the option to input their Twitter account name to connect the application to their own account. With this, they can view more personalized data, such as data only including their network of followers that is located within the Network Pages.

Helpful Tips

Locations

Zooming into the Map Visualization: Within the Locations analysis page you can zoom in on an individual U.S state by clicking on it. Clicking again will zoom back out to the whole map.

Time

Analyzing Trends Over Time: Within the Time analysis page you can add different trending hashtags within the selections pane. You may add up to five different trends. You can also select a given time interval in Days. Clicking update will update the graph and reveal the visualization.

Network

Analyzing a Users Network: The Word CLoud page analyzes a given users network for trending keywords and topics. Enter a users twitter account, then select the number of accounts that you wish to search within that users network and then click update graph.
